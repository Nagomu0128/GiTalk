# 018 - ランディングページ リデザイン

## 概要

`1.png` のデザインを参考に、ランディングページ (`frontend/app/page.tsx`) を再デザインした。

## 変更内容

### 画像素材の追加
以下のファイルを `frontend/public/` に配置:
- `logo-dark.png` (2.png): ダーク背景用ロゴ
- `logo-light.png` (3.png): ライト背景用ロゴ（ヘッダー・ヒーローで使用）
- `icon-branch.png` (4.png): 分岐アイコン（回路基板デザイン）
- `icon-merge.png` (5.png): 統合アイコン（2ブランチが合流するデザイン）
- `icon-save.png` (6.png): 保存アイコン（箱＋下矢印）

### page.tsx の変更
- **ヘッダー**: テキストのみ → `logo-light.png` + "GiTalk" テキスト。「はじめる」ボタンを青色からダーク（`bg-gray-900`）に変更
- **ヒーロー**: 大きな `logo-light.png` を中央に追加。CTAボタンのテキストを「無料ではじめる」→「会話を始める →」に変更、スタイルを pill 型（`rounded-full`）ダークボタンに変更
- **機能紹介**: 絵文字 → 実画像アイコン。分岐・統合はダーク背景（`bg-gray-900`）のボックス内に表示、保存はボーダーボックスで表示
