# 11 - ワイヤーフレーム

## 概要

GiTalk の全画面のワイヤーフレーム定義。
各画面の構成要素、インタラクション、状態遷移を明確にする。

## 画面一覧とルーティング

| ルート | 画面名 | 認証 | MVP |
|--------|--------|------|-----|
| `/` | ランディングページ | 不要 | Yes |
| `/login` | ログイン/サインアップ | 不要 | Yes |
| `/dashboard` | ダッシュボード | 必須 | Yes |
| `/dashboard/conversations` | 会話一覧 | 必須 | Yes |
| `/dashboard/repositories` | リポジトリ一覧 | 必須 | Yes |
| `/conversation/:id` | 会話画面（メイン） | 必須 | Yes |
| `/repository/:id` | リポジトリ詳細 | 条件付き | Yes |
| `/explore` | 公開リポジトリ探索 | 不要 | No |
| `/settings` | ユーザー設定 | 必須 | No |

| `RepositoryListItem` | リポジトリ一覧 | リスト形式のリポジトリ行 |

| `TreeNode` | TreeView 内 | カスタムノードコンポーネント |
| `SummaryNode` | TreeView 内 | マージ要約ノード（破線） |
| `ChatView` | 会話画面 | メッセージ一覧 + ストリーミング表示 |
| `MessageBubble` | ChatView 内 | 個別メッセージ（ユーザー/AI） |
| `MessageInput` | 会話画面 | テキスト入力 + モデル選択 + 送信 |
| `BranchSelector` | 会話画面ヘッダー | ブランチ切替ドロップダウン |
| `NodeContextMenu` | TreeView 内 | 右クリックメニュー |
| `MergeDialog` | 会話画面 | マージ設定モーダル |
| `PushDialog` | 会話画面 | push 設定モーダル |
| `DiffView` | 会話画面 | 左右分割の差分表示 |
| `CommandPalette` | 全画面 | Ctrl+K コマンドパレット |
| `MiniTreePreview` | カード内 | 小さいツリー図（SVG） |
